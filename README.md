# 📚 대전 공공도서관 도서 검색기

대전 시내 공공도서관의 **대출 가능 여부를 한 번에 확인**할 수 있는 도서 검색 도구입니다.  
원하는 책을 검색어로 입력하면, 어느 도서관에 해당 책이 있고 지금 바로 빌릴 수 있는지 알려줍니다.

---

## 어떤 도서관을 지원하나요?

대전 5개 구(서구·유성구·동구·중구·대덕구)의 공공도서관 **24곳**을 지원합니다.

| 구 | 도서관 |
|---|---|
| 서구 | 월평, 둔산, 가수원, 갈마, 서구어린이 |
| 유성구 | 관평, 구암, 구즉, 노은, 아가랑, 원신흥, 유성, 유성엑스포, 전민, 진잠 |
| 동구 | 가오, 무지개, 용운, 자양, 판암, 홍도 |
| 중구 | 대전학생교육문화원, 산성어린이, 한밭 |
| 대덕구 | 석봉, 송촌, 안산 |

---

## 사용 방법

### 방법 1 — 웹 브라우저로 사용하기 (Streamlit 앱, 권장)

1. 아래 **설치** 단계를 완료합니다.
2. 터미널에서 다음 명령어를 실행합니다.

   ```
   streamlit run streamlit_app.py
   ```

3. 브라우저가 자동으로 열리면 (보통 `http://localhost:8501`):
   - **키워드** 란에 찾고 싶은 책 제목이나 주제어를 한 줄에 하나씩 입력합니다.
   - 원하는 **도서관**을 선택하거나, 선택하지 않으면 전체 도서관에서 검색합니다.
   - **🔍 검색** 버튼을 누릅니다.
   - 결과 화면에서 도서관별 대출 가능 도서 목록을 확인합니다.
   - **"대출 가능 도서만 보기"** 체크박스로 결과를 바로 필터링할 수 있습니다.

### 방법 2 — 웹 브라우저로 사용하기 (FastAPI 앱)

1. 아래 **설치** 단계를 완료합니다.
2. 터미널에서 다음 명령어를 실행합니다.

   ```
   python app.py
   ```

3. 브라우저에서 `http://127.0.0.1:8000` 을 엽니다.
4. 키워드를 입력하고 도서관을 선택한 뒤 **검색** 버튼을 누릅니다.

### 방법 3 — 명령줄(터미널)에서 사용하기

터미널에서 아래처럼 실행합니다.

```
# 키워드만 지정 (전체 도서관 검색)
python main.py --key 파이썬

# 키워드 여러 개
python main.py --key 파이썬 "데이터 분석"

# 특정 도서관 지정
python main.py --key 파이썬 --lib 한밭도서관
```

결과는 도서관별 대출 가능 권수 순으로 출력됩니다.

## 설치 방법

### 필요한 것

- Python 3.10 이상
- 인터넷 연결 (도서관 사이트에서 정보를 가져옵니다)

### 설치 순서

```
# 1. 이 저장소를 내려받습니다
git clone <저장소 주소>
cd dj_library

# 2. 필요한 패키지를 설치합니다
pip install -r requirements.txt
```

## 자주 묻는 질문

**Q. 검색 결과가 오래 걸려요.**  
A. 키워드·도서관 조합마다 최대 5페이지씩 인터넷 요청을 보내기 때문에, 조합이 많을수록 시간이 걸립니다. 도서관을 좁혀서 검색하면 빠릅니다.

**Q. 특정 도서관이 목록에 없어요.**  
A. 현재 대전시 공공도서관 24곳만 지원합니다. 다른 지역이나 사립 도서관은 지원하지 않습니다.

**Q. "정보 없음"은 무슨 뜻인가요?**  
A. 도서관 사이트에서 대출 상태 정보를 제공하지 않은 경우입니다. 직접 도서관에 문의해 주세요.
